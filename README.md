# Nautiflux

Nautiflux est une application media player moderne, fluide et responsive, conçue pour offrir une expérience utilisateur inspirée des plateformes comme Netflix ou Disney+.

Ce projet a été réalisé dans le cadre d'un cours de **Master 1 (M1)** portant sur **Tailwind CSS**. Il a été réalisé par **Louis CONSTANT** et **Corentin GUILLAUME**.

## Fonctionnalités Clés

- **Design Premium & Responsive** : Interface entièrement conçue avec Tailwind CSS v4, s'adaptant du mobile au desktop via des grilles dynamiques et des composants adaptatifs.
- **Dark/Light Mode** : Possibilité de changer entre un thème Dark et un thème Light.
- **Barre de Recherche Throttle** : Recherche instantanée de films par titre optimisée par un système de *throttle* pour une meilleure performance.
- **Lecteur Vidéo Avancé** : Lecture de contenus vidéo avec sauvegarde automatique de la progression. Si vous rafraîchissez la page, la vidéo reprend exactement là où vous vous étiez arrêté.
- **Navigation Fluide** : Utilisation de `Vue Router` pour une expérience Single Page Application (SPA) sans rechargement de page.

## Configuration et Installation

### 1. Cloner le repository
```bash
git clone https://github.com/HarleZe/Nautiflux.git
cd Nautiflux
```

### 2. Installer les dépendances
```bash
npm install
```

### 3. Lancer le serveur de développement
```bash
npm run dev
```

Une fois lancé, l'application sera disponible sur `http://localhost:5173`.
